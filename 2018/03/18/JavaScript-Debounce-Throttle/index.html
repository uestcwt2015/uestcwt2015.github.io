函数去抖（Debounce）

对于那些频繁触发的事件，我们期望在某段时间內只触发一次，而不是每次触发都要执行回调

简单的说就是我们忽略掉多次触发的相同事件，只关注某一次触发

自己的实现：

原理： 通过闭包存储timer，当连续触发事件时，上一次的定时器还未执行回调就会被下一次触发给清掉，所以只会最后一次触发才会执行回调
